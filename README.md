**Project Title**
AI-Based Food Calorie Calculator
**Problem Statement**
People often find it difficult to estimate the calorie content of food items from images. This project aims to build an AI-powered system that identifies food items from images and estimates their calorie values.
**Dataset**
Food image dataset.
Nutritional calorie reference dataset.
**Technologies Used**
Python
Streamlit
Gemini API
Computer Vision
NumPy
**Model Used**
AI image recognition using Gemini API.
**Results**
Successfully identifies food items from images.
Displays estimated calorie values.
Provides a user-friendly web interface.
